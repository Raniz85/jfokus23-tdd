TDD Katas

Welcome to Raniz's collection of TDD katas. The following katas are what I use when doing TDD workshops and are developed with the intention to be suitable for practicing test driven development.

Layout

Each kata consists of a series of assignments where you have to solve the current assignment before the next is revealed.

The next part is encrypted with the solution of the assignment as the passphrase, so there's no peeking ahead!

Katas

  • CEO Bowling - A bunch of CEOs goes bowling and need help with the scoring
  • Test Flight - A malfunctioning autopilot and a hot tub, why not?
  • Dice Game - Implement a dice game that gets progressively more complex

Source Code

The source code is available at github.com/raniz85/tdd-katas. Be aware that the source code contains the correct solutions so you might want to solve the assignments before peeking around.
